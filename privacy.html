Politique de Confidentialité – Hope RP Vote Dernière mise à jour : 2025-09-09 Hope RP Vote est une extension Chrome qui aide l’utilisateur à se rappeler de voter à nouveau pour son serveur sur Top-Serveurs en affichant un compte à rebours et en envoyant des notifications. Collecte et traitement des données Nous ne collectons, n’envoyons ni ne partageons aucune donnée personnelle vers des serveurs externes. Les informations suivantes peuvent être stockées localement dans votre navigateur via l’API chrome.storage : le pseudo que vous saisissez pour voter ; la date/heure de votre dernier vote (pour le rappel). Ces informations restent sur votre appareil et sont utilisées uniquement pour faire fonctionner l’extension. Permissions utilisées storage : enregistrer localement le pseudo et l’heure du dernier vote ; alarms : planifier les rappels (notifications) à intervalles définis ; notifications : afficher les rappels quand vous pouvez revoter ; tabs : ouvrir la page de vote lorsque vous cliquez sur « Voter maintenant » ; Autorisations d’accès à l’hôte L’extension déclare un accès limité à l’hôte : https://top-serveurs.net/*. Cela sert uniquement à ouvrir la page de vote correspondante lorsque vous choisissez d’y accéder depuis l’extension. Contrôle par l’utilisateur Effacer les données locales : vous pouvez réinitialiser ou supprimer les données stockées via le menu de l’extension ou en supprimant l’extension de Chrome ; Révocation : vous pouvez désinstaller l’extension à tout moment (« Gérer les extensions ») ; Consentement : l’usage de l’extension vaut consentement à cette politique. Partage et vente de données Nous ne vendons pas et ne partageons pas de données utilisateurs avec des tiers. Sécurité Le code de l’extension s’exécute localement et n’inclut pas de scripts distants.
