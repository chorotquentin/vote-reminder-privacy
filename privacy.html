Politique de Confidentialité – Vote Reminder

Vote Reminder est une extension Chrome conçue pour faciliter le vote sur Top Serveur GTA en rappelant à l’utilisateur quand il peut voter à nouveau.

Collecte de données

Cette extension ne collecte aucune donnée personnelle. Toutes les informations, comme :

  • le pseudo que vous utilisez pour voter
  • la date et l’heure de votre dernier vote

sont stockées localement dans votre navigateur via l’API chrome.storage. Ces informations ne sont jamais envoyées à un serveur externe et ne sont partagées avec personne.

Permissions utilisées

L’extension utilise les permissions suivantes :

  • storage : pour stocker localement le pseudo et le dernier vote.
  • notifications : pour envoyer des rappels lorsque vous pouvez voter à nouveau.
  • alarms : pour planifier les notifications de rappel.
  • tabs : pour ouvrir la page de vote dans un nouvel onglet lorsque vous cliquez sur “Voter maintenant”.

Sécurité

Tout le code JavaScript et HTML de l’extension est local et ne dépend d’aucun script distant. Cela garantit que vos données restent sécurisées et privées.
