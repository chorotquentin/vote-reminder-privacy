Politique de Confidentialité - Hope RP Vote

Dernière mise à jour : 2025-09-09

Introduction

Hope RP Vote est une extension Chrome qui aide l’utilisateur à se rappeler de voter sur son serveur Hope RP via Top-Serveurs en affichant un compte à rebours et en envoyant des notifications.

Collecte et traitement des données

Nous ne collectons, n’envoyons ni ne partageons aucune donnée personnelle vers des serveurs externes. Les informations suivantes sont uniquement stockées localement dans votre navigateur :

  • Le pseudo que vous saisissez pour voter,
  • La date/heure de votre dernier vote (pour calculer le rappel).

Permissions utilisées

  • storage : enregistrer localement le pseudo et l’heure du dernier vote,
  • alarms : programmer les rappels automatiques,
  • notifications : afficher une notification lorsque vous pouvez revoter.

Contrôle par l’utilisateur

  • Vous pouvez supprimer vos données locales via le menu de l’extension,
  • Vous pouvez désinstaller l’extension à tout moment,
  • L’utilisation de l’extension vaut consentement à cette politique.

Sécurité

Le code de l’extension s’exécute localement dans le navigateur. Aucune transmission de données personnelles n’est effectuée.

Partage et vente de données

Nous ne vendons pas et ne partageons pas les données utilisateurs avec des tiers.

Privacy Policy - Hope RP Vote

Last updated: 2025-09-09

Introduction

Hope RP Vote is a Chrome extension that helps users remember to vote for their Hope RP server on Top-Serveurs by showing a countdown timer and sending notifications.

Data collection and processing

We do not collect, send, or share any personal data to external servers. The following information is only stored locally in your browser:

  • The nickname you enter for voting,
  • The date/time of your last vote (to calculate reminders).

Permissions used

  • storage: to save locally your nickname and last vote time,
  • alarms: to schedule automatic reminders,
  • notifications: to display a notification when you can vote again.

User control

  • You can delete your local data via the extension menu,
  • You can uninstall the extension at any time,
  • Using the extension constitutes acceptance of this policy.

Security

The extension’s code runs locally in the browser. No personal data is transmitted.

Data sharing and selling

We do not sell or share user data with third parties.
